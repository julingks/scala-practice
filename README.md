# Scala Practice

## Install on Mac OS X

```
$ brew install scala
```

## Vim Syntax Highlighting

* [https://github.com/derekwyatt/vim-scala](https://github.com/derekwyatt/vim-scala)

Add the following to .vimrc:

```
Plugin 'derekwyatt/vim-scala'
```

and then run

```
:PluginInstall
```
